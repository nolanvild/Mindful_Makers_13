# Mindful_Makers_13
This is the repo that I have made for the 13th Mindful Makers hackathon

## Running locally

This project uses [uv](https://docs.astral.sh/uv/) to manage the local dev server. The app must be served over `http://localhost` (not opened as a `file://` path) for the browser to grant camera access.

```sh
uv run main.py
```

This starts a static file server at http://127.0.0.1:8000/index.html and opens it in your browser.
